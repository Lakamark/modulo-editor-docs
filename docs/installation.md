# Installation
Install ModuloEditor using your preferred package manager.

::: code-group

```bash [npm]
npm install @lakamark/modulo-editor
```
```bash [pnpm]
pnpm add @lakamark/modulo-editor
```
```bash [yarn]
yarn add @lakamark/modulo-editor
```
```bash [bun]
bun add @lakamark/modulo-editor
```
:::

## Requirements

- Modern browser with ES Modules support
- TypeScript 5.x or later (recommended)

## Next Step
