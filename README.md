# ModuloEditor Documentation

Official documentation website for ModuloEditor.

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run docs:dev
```

Build the documentation:

```bash
npm run docs:build
```

Preview the production build:

```bash
npm run docs:preview
```

## Stack

* VitePress
* TypeScript
* Vue
