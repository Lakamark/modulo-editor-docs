Official documentation for **ModuloEditor**.

ModuloEditor is a modular, extensible and safe Markdown editor with plugin architecture and textarea form integration.

# Development

Install dependencies

```bash
npm install
```

## Start dev server:
```bash
npm dev
```

## Build documentation:
```bash
npm build
```

## Preview production build:
```bash
npm run preview
```
